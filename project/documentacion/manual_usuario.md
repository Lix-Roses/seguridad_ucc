# Manual de Usuario

## Introducción

Este sistema permite gestionar usuarios y mejorar la seguridad mediante autenticación segura.

---

## Objetivo del sistema

El sistema fue desarrollado para administrar usuarios y proteger el acceso mediante validación segura de credenciales.

---

## Inicio de sesión

El usuario debe ingresar:

- correo electrónico
- contraseña

El sistema validará las credenciales antes de permitir el acceso.

---

## Registro de usuarios

El sistema permite registrar nuevos usuarios mediante un formulario.

Datos solicitados:
- nombre
- correo electrónico
- contraseña

---

## Seguridad

Las contraseñas son almacenadas utilizando técnicas de hash para proteger la información de los usuarios.

---

## Auditoría

El sistema registra acciones importantes realizadas dentro de la plataforma.

---

## Tecnologías utilizadas

- Python
- PostgreSQL
- GitHub
- Alembic

---

## Recomendaciones

- Utilizar contraseñas seguras.
- No compartir credenciales.
- Mantener actualizada la base de datos.

---

## Conclusión

El sistema proporciona una solución básica de seguridad y administración de usuarios para el proyecto académico.
